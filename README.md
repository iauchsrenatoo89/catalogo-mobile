# Catálogo Mobile

Aplicativo desenvolvido com React Native e Expo.

## Funcionalidades

- Login
- Navegação entre categorias
- Consumo de API
- Lista de produtos
- Tela de detalhes
- Logout

## Tecnologias

React Native  
Expo  
Expo Router  
Axios  

## Como executar o projeto

1. Instale as dependências:

npm install

2. Inicie o projeto com Expo:

npx expo start --tunnel -c

3. Abra o aplicativo Expo Go no celular e escaneie o QR Code gerado.

### Observação
Durante os testes, a execução com o comando `npx expo start --tunnel -c` apresentou maior estabilidade no ambiente utilizado, garantindo a correta conexão com o dispositivo móvel via Expo Go.
